## 关键问题
1. 如何平衡主动复盘推送的频率以避免用户疲劳？
2. 在共识模式下，哪些模型对识别英语语法细微差别最有效？

## 决策记录
| 决策 | 理由 |
|------|------|
| 使用 SQLite 进行持久化 | 提供可靠的、支持异步的对话历史和用户状态存储。 |
| 实现 _clean_reply 正则拦截 | 通过剔除多余的 Markdown/表情符号，确保 UI 的一致性。 |
| 限制推送时间 (12:00-22:00) | 尊重用户休息时间，避免深夜骚扰。 |

## 遇到的错误
| 错误 | 尝试次数 | 解决方法 |
|-------|---------|------------|
| telegram.error.Conflict | 1 | 终止了多余的 Bot 进程并重启服务。 |
| ModuleNotFoundError: aiosqlite | 1 | 使用 --break-system-packages 安装了 aiosqlite。 |
